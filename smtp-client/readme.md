# Web Proxy

## Описание
SMTP клиент. В конфигурационном файле пользователь задает адрес получателя(лей), 
тему (возможно на русском языке) и имена файлов-аттачментов для отсылки в виде вложения.

## Запуск
```
python3 smtpclient.py -i client.ini
```
